# Positional Hierarchy

An interactive BJJ (Brazilian Jiu-Jitsu) positional hierarchy chart built with Next.js. Visualizes positions from best to worst with clickable techniques that open embedded YouTube tutorial videos.

## Features

- **Tiered position chart** — Positions organized into BEST, Neutral, and WORST tiers with color-coded borders (green, orange, red)
- **Clickable techniques** — Each technique links to a YouTube video that plays in an embedded popup
- **Sub-menus** — Some techniques expand into multiple video options (e.g., defensive techniques from guard top)
- **Position images** — Visual reference for each position

## Positions Covered

| Tier | Positions |
|------|-----------|
| BEST | Back control, Mount top, Side control top, Guard bottom |
| Neutral | 1/2 Guard top, 1/2 Guard bottom, Guard top |
| WORST | Side control bottom, Mount bottom, Back taken |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
